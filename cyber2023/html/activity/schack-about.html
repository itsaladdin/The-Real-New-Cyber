Schack

○ Kungen rör sig en ruta i vilken riktning som helst.
○ Drottning rör sig över flera rutor i alla ritningar.
○ Törnet rör sig vertikalt eller horisontellt över flera rutor.
○ Löparen rör sig diagonalt över flera rutor.
○ Inga särskilda drag,du kan undvika att använda specialdrag som rockade och bonde omvandling för att göra spelet enklare.
○ Ingen tidsbegränsning,du kan spela utan att använda en schackklocka och kan ta så mycket tid som du vill för att överväga dina drag.
